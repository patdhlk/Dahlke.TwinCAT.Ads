# Contributing

Thank you for your interest in contributing to Dahlke.TwinCAT.Ads!

## Getting Started

1. Fork and clone the repository
2. Open `Dahlke.TwinCAT.Ads.sln` in your IDE
3. Run the tests: `dotnet test`

## Development

- This project targets .NET 8.0, 9.0, and 10.0
- Run `dotnet build` to verify your changes compile across all frameworks
- Run `dotnet test` to verify all tests pass
- Follow the existing code style (enforced by `.editorconfig`)

## Pull Requests

- Create a feature branch from `main`
- Keep changes focused — one feature or fix per PR
- Add or update tests for any changed behavior
- Ensure CI passes before requesting review

## Reporting Issues

Open an issue on GitHub with:
- A clear description of the problem or suggestion
- Steps to reproduce (for bugs)
- Your .NET version and OS
